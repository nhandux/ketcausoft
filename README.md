## Step to install

Repository: [https://github.com/nhandux/gulpfile.git](https://github.com/nhandux/gulpfile.git)

1. Clone from heystars repository
```bash
git clone https://github.com/nhandux/gulpfile.git
```
2. Checkout to api branch (this is common branch for gulp file)
```bash
git checkout develop
```
5. Generate application key and clear config default
```bash
- yarn insall
```
